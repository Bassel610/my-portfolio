# Basel Sherif — Developer Portfolio

Personal developer portfolio with glassmorphism UI, wheel-based navigation, scroll-driven animations, and SEO optimization.

**Live demo:** https://basels-portfolio.netlify.app

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Material-UI 7
- Framer Motion
- react-intersection-observer

## Getting Started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
npm start
```
