## 🔍 Overview

This is my personal developer portfolio — a polished, animated single-page experience built with **Next.js 15** and **React 19**. The design centers on a glassmorphism aesthetic, a custom wheel-based navigation pattern, and motion-driven interactions powered by Framer Motion.

## ✨ Features

- 🪟 **Glassmorphism UI** — frosted-glass surfaces with depth and blur effects
- 🎡 **Wheel-based navigation** — a unique radial nav for switching sections
- 🎬 **Scroll-driven animations** with Framer Motion + react-intersection-observer
- ⚡ **Next.js App Router** for fast, SEO-friendly server-rendered pages
- 📱 **Fully responsive** across mobile, tablet, and desktop breakpoints
- 🔍 **SEO optimization** with metadata API and semantic markup
- 🎨 **Material-UI 7** design tokens for consistent styling
- 🚨 **Error handling** with graceful fallback UIs

## 🛠️ Tech Stack

**Framework**
- Next.js 15 (App Router)
- React 19

**UI & Styling**
- Material-UI 7 (`@mui/material`, `@mui/icons-material`)
- Emotion (CSS-in-JS)
- Custom glassmorphism utilities

**Animation**
- Framer Motion 11
- react-intersection-observer

## 🚀 Getting Started

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Installation

```bash
git clone https://github.com/Bassel610/my-portfolio.git
cd my-portfolio
npm install
```

### Development

```bash
npm run dev          # http://localhost:3000
```

### Production Build

```bash
npm run build        # Build for production
npm start            # Run the production server
```
